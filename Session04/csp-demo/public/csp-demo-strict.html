CSP Demo - Strict Policy

Strict CSP Policy Demonstration

This page implements a strict CSP policy that demonstrates multiple security features:

  • Only allows scripts from same origin and cdn.jsdelivr.net
  • Allows inline styles (needed for this demo)
  • Restricts images to same origin and example.com
  • Blocks all object/embed/applet content
  • Prevents site from being framed by other sites
  • Restricts form submissions to same origin

Current CSP Policy

Content-Security-Policy: 
    default-src 'self';
    script-src 'self' https://cdn.jsdelivr.net;
    style-src 'self' 'unsafe-inline';
    img-src 'self' https://*.example.com;
    font-src 'self';
    connect-src 'self';
    frame-ancestors 'none';
    form-action 'self';
    base-uri 'self';
    object-src 'none'
