What is Content Security Policy?

Content Security Policy (CSP) is a security feature that helps prevent various types of attacks, including Cross-Site Scripting (XSS) and data injection attacks. It works by restricting the sources from which content can be loaded and executed.

Key CSP Directives:

  • default-src: Default fallback for other fetch directives
  • script-src: Controls which scripts can be executed
  • style-src: Controls which styles can be applied
  • img-src: Controls which images can be loaded
  • connect-src: Controls which URLs can be loaded using script interfaces
